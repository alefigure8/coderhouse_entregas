# Carrera Backend

Entregas de trabajos de la carrera Backend de Coderhouse.

## Configuración

### Clonar

```js
git clone https://github.com/alefigure8/coderhouse_entregas.git
```

### Instalar

```ps1
npm install
```

### Ejecutar

```js
npm run dev
```
